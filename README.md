# docker-scout-cli

Proyecto académico para demostrar DevOps, CI/CD, Docker Scout CLI y mantenimiento de software.

## Objetivo
Crear un repositorio nuevo con automatización, pruebas, contenedores y escaneo de vulnerabilidades.

## Tecnologías
- Go
- GitHub Actions
- Docker
- Docker Scout CLI
- Visual Studio Code

## Estructura
- cmd/
- internal/
- tests/
- scripts/
- docs/
- .github/workflows/

## Uso
Primero se desarrolla, luego se prueban los comandos, después se automatiza con GitHub Actions y finalmente se valida la imagen con Docker Scout CLI.
